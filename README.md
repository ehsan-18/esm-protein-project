# 🧬 ESM-based Protein Engineering Mini Project

This project demonstrates a simple protein engineering workflow using Python and Meta AI's ESM (Evolutionary Scale Modeling) in Google Colab.

---

## 🎯 Project Goal

To analyze protein sequences, simulate mutations, and study their properties using computational biology tools and protein language models.

---

## 🧪 Features

- Protein sequence analysis
- Point mutation and mutation scanning
- Sequence entropy calculation
- Hydrophobicity analysis
- ESM embeddings (via Colab notebook)
- Mutation effect comparison

---

## 🧬 Tools Used

- Python 3
- PyTorch
- ESM (Meta AI protein language model)
- Google Colab

---
